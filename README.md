# OECExtension

This mod is a community mod for the Overwatch Expeditionary Corps in the game ArmA 3. The repository contains modifications to ArmA 3 config and seeks to specialize the game experience for members of the OEC community.

Any inquiries about this mod can be directed to dirt0503 or acolytepetee over Discord. To see the OEC community, follow the link here. https://discord.gg/DX7PKNT54G
